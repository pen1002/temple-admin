# 카카오 알림톡 템플릿 신청 양식

Solapi 콘솔(https://console.solapi.com)에서 아래 3개 템플릿을 등록합니다.

---

## 1. 기도/공양 접수 알림 (OFFERING)

**템플릿 이름**: 기도공양접수알림
**카테고리**: 업데이트 정보

```
[#{사찰명} #{유형}]

#{이름} 불자님이 #{유형}에 동참하였습니다.

발원: #{발원}

감사합니다. 나무아미타불 🙏
```

---

## 2. 신도 등록 알림 (SIDO)

**템플릿 이름**: 신도등록알림
**카테고리**: 업데이트 정보

```
[#{사찰명} 신도등록]

#{이름}님이 신도로 등록되었습니다.

법명: #{법명}
연락처: #{연락처}

나무아미타불 🙏
```

---

## 3. 위패 봉안 알림 (MEMORIAL)

**템플릿 이름**: 위패봉안알림
**카테고리**: 업데이트 정보

```
[#{사찰명} 위패봉안]

#{이름}님이 영가님의 위패를 봉안하였습니다.

발원: #{발원}

나무아미타불 🙏
```

---

## Solapi 설정 순서

### 1단계: 카카오톡 채널 연동
1. Solapi 콘솔 → 카카오톡 → 채널 관리
2. 카카오 비즈니스 채널 검색 → 연동
3. 연동 완료 후 PFID(채널 ID) 확인

### 2단계: 알림톡 템플릿 등록
1. Solapi 콘솔 → 카카오톡 → 알림톡 템플릿
2. 위 3개 템플릿 등록
3. 카카오 검수 제출 (1~2 영업일 소요)
4. 승인 후 각 템플릿 ID 확인

### 3단계: Vercel 환경변수 추가
```
SOLAPI_PFID=KSxxxxxx          # 카카오 채널 ID
SOLAPI_TPL_OFFERING=KA0xxxxx  # 기도/공양 템플릿 ID
SOLAPI_TPL_SIDO=KA0xxxxx      # 신도등록 템플릿 ID
SOLAPI_TPL_MEMORIAL=KA0xxxxx  # 위패봉안 템플릿 ID
```

### 동작 방식
- PFID + 템플릿ID 설정 시 → **알림톡 우선 발송**
- 알림톡 실패 시 → **자동 SMS 대체**
- PFID 미설정 시 → **SMS만 발송** (현재 상태)
